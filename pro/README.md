## Quick start

Drop `neumorph-pro.css` after `styles.css`:
```html
<link rel="stylesheet" href="styles.css" />
<link rel="stylesheet" href="neumorph-pro.css" />
```

React:
```tsx
import { Button, Card, Toggle, Modal } from './react/Neumorph';
<Button gold>Pay now</Button>
```
